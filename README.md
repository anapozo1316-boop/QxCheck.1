# 🩺 QxCheck

Software básico para la verificación de equipos quirúrgicos.

## Funciones

- Selección de equipos quirúrgicos
- Visualización del instrumental
- Comparación de cantidades requeridas y disponibles
- Detección automática de faltantes
- Tabla editable

## Tecnologías utilizadas

- Python
- Pandas
- Streamlit
- Google Colab
- GitHub
