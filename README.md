# 🩺 QxCheck

QxCheck es una aplicación desarrollada en Python y Streamlit para la verificación básica de equipos quirúrgicos.

El sistema permite:

- Visualizar instrumental quirúrgico
- Comparar cantidades requeridas y disponibles
- Detectar faltantes
- Validar equipos quirúrgicos de forma rápida y visual

## Tecnologías utilizadas

- Python
- Pandas
- Streamlit
- Google Colab
- GitHub

## Objetivo

Facilitar el control básico de instrumental quirúrgico en centrales de esterilización y áreas quirúrgicas.
