# 🩺 QxCheck

Software de verificación de equipos quirúrgicos desarrollado en Python y Streamlit.

## Funciones

- Crear nuevos equipos quirúrgicos
- Agregar instrumental
- Editar cantidades
- Detectar faltantes
- Validar equipos quirúrgicos

## Tecnologías

- Python
- Pandas
- Streamlit
- Google Colab
- GitHub
